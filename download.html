About Zyphor V1.0.0

Zyphor V1.0.0 was built using AppMySite as a temporary solution to make the Android app available as soon as possible. While it provides the core Zyphor experience, it is not the final version of the app.

The official native Zyphor application is currently being developed by Team Vaprix. Future releases will be built entirely by Team Vaprix and will feature a redesigned interface, significantly improved performance, additional features, and a much better overall experience.
